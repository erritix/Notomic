I made this cuz theirs suck on some aspects. Starting from performance, feature, extensibility, interface, and aislop

## Features

- [x] Based on Markdown
- [x] Performance (achiving it using Svelte and Wails)
- [ ] Synchronization (using custom program)
- [x] Categorization with Folder
- [ ] Labeling with Tags system
